# 钓点探秘 · 桌面网页版

> 把那些只属于你的好位置,留在地图上。

一个钓鱼地点记录 / 分享应用的桌面端网页版原型 —— 原本是微信小程序形态,
小程序开发环境配置太繁琐,先用网页版看效果。

## 怎么看效果

dev server 已经跑起来了,浏览器直接打开:

```
http://localhost:5174
```

如果没起来,在项目目录跑:

```bash
npm install
npm run dev
```

## 现在有什么

### 4 个页面

| 页面 | 路由 | 说明 |
|------|------|------|
| 🗺️ 探秘地图 | `/` | 主视图。手绘风格中国地图 + 8 处钓点标记,左侧筛选卡片列表,点标记弹详情卡 |
| 📍 钓点详情 | `/spot/:id` | 笔记、照片画廊、鱼种卡片、最佳时间、附近推荐、收藏/分享 |
| ➕ 新增钓点 | `/add` | 4 步表单:基础信息 → 位置(点地图选点)→ 鱼种/时间 → 笔记封面 |
| 🎒 我的钓点 | `/profile` | 头部成就卡 + 4 个数据指标 + 收藏/我的/统计三个 tab |

### 视觉

- 户外探险杂志风:奶油底 `#f4ede0` + 苔藓绿 `#3d5a47` + 暖橙 `#c8553d`
- 标题用 Noto Serif SC 衬线字体,正文 Inter
- 手绘 SVG 中国地图(山脉符号、河流、海洋波浪、装饰罗盘、比例尺)
- 8 条精心设计 mock 数据,涵盖湖泊/江河/水库/近海

### 技术

- Vue 3 + Composition API + `<script setup>`
- Vite 5 dev server
- vue-router 4(hash 模式)
- 纯 CSS(无 UI 库),自定义设计系统
- LocalStorage 持久化(刷新不丢)

## 试一下这几个交互

1. **筛选** —— 主页左上角切水域(湖泊/江河/水库/近海)
2. **点标记** —— 地图上点圆形标记,右侧弹详情卡 → "查看详情" 进详情页
3. **新增** —— 顶部"新增钓点",第二步点击地图选位置,提交后会跳到详情页
4. **收藏** —— 详情页点 "☆ 收藏钓点",然后到 "我的钓点 → 收藏的" 看
5. **持久化** —— 刷新页面,你的数据还在
6. **重置** —— 个人中心右上角"重置示例数据",回到初始状态
